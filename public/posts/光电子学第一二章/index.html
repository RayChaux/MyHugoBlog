第1章 电磁理论及拓展
#

1.1 电磁相互作用简介
#

1.1.1 四大基本相互作用
#

自然界存在四大基本相互作用，电磁相互作用是其中之一，广泛存在于宇宙中（如电子运动、光子传播等）：

  • 引力相互作用
  • 电磁相互作用
  • 弱核相互作用（β衰变相关）
  • 强核相互作用

1.1.2 电磁相互作用的普遍性
#

电磁相互作用贯穿于微观（电子、光子相互作用）和宏观（电磁波传播、电磁设备工作）场景，是光电子学的核心相互作用基础。

1.2 麦克斯韦方程组与电磁统一
#

1.2.1 麦克斯韦方程组的形式
#

名称 积分形式 微分形式
高斯定律（电场） $\iint_{\partial \Omega} \vec{D} \cdot d\vec{S} = \iiint_{\Omega} \rho_f dV$ $\nabla \cdot \vec{D} = \rho_f$
高斯定律（磁场） $\iint_{\partial \Omega} \vec{B} \cdot d\vec{S} = 0$ $\nabla \cdot \vec{B} = 0$
法拉第电磁感应定律 $\oint_{\partial \Sigma} \vec{E} \cdot d\vec{\ell} = -\frac{d}{dt} \iint_{\Sigma} \vec{B} \cdot d\vec{S}$ $\nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t}$
安培-麦克斯韦定律 $\oint_{\partial \Sigma} \vec{H} \cdot d\vec{\ell} = \iint_{\Sigma} \vec{J}_f \cdot d\vec{S} + \frac{d}{dt} \iint_{\Sigma} \vec{D} \cdot d\vec{S}$ $\nabla \times \vec{H} = \vec{J}_f + \frac{\partial \vec{D}}{\partial t}$

1.2.2 各定律核心解释
#

  1. 高斯定律（电场）：电场通量与闭合曲面内的自由电荷成正比，源于库仑定律：$|\vec{E}| = \frac{1}{4\pi\varepsilon_0} \frac{|q|}{r^2}$（点电荷电场强度公式）。
  2. 高斯定律（磁场）：磁场通量始终为零，表明磁场是无源场（无磁单极子），满足$\vec{B} = \nabla \times \vec{A}$（$\vec{A}$为矢量势）。
  3. 法拉第电磁感应定律：变化的磁场产生电场，电场的旋度等于磁感应强度的时间变化率的负值，反映磁生电的规律。
  4. 安培-麦克斯韦定律：磁场的旋度由传导电流和位移电流共同产生，位移电流$\vec{J}_D = \frac{\partial \vec{D}}{\partial t}$是麦克斯韦的关键补充。

1.2.3 位移电流的引入原因
#

  • 核心矛盾：无位移时，$\nabla \cdot \vec{J} = \nabla \cdot (\nabla \times \vec{H}) = 0$，结合电荷守恒$\nabla \cdot \vec{J} + \frac{\partial \rho}{\partial t} = 0$，会推出$\frac{\partial \rho}{\partial t} = 0$（无电荷积累），与实际矛盾（如电容器充电时极板电荷积累）。
  • 实例：电容器充电时，极板间无传导电流，但存在电场变化$\frac{\partial \vec{D}}{\partial t}$，位移电流$I_D = \frac{dQ}{dt} = S \varepsilon_0 \frac{\partial D}{\partial t}$（$S$为极板面积），保证电流连续性。

1.2.4 麦克斯韦方程组的意义
#

  • 统一电场和磁场，证明变化的电场产生磁场、变化的磁场产生电场，预言电磁波的存在。
  • 真空中电磁波速度：$v = c = \frac{1}{\sqrt{\mu_0 \varepsilon_0}} = 299792458 , \text{m/s}$（光速）。

1.2.5 磁单极子与超材料
#

  • 磁单极子：目前未实验证实，若存在，麦克斯韦方程组需修正（见下表）。
    无磁单极子 有磁单极子（韦伯规范） 有磁单极子（安培-米规范）
    $\nabla \cdot \vec{B} = 0$ $\nabla \cdot \vec{B} = \rho_m$ $\nabla \cdot \vec{B} = \mu_0 \rho_m$
    $-\nabla \times \vec{E} = \frac{\partial \vec{B}}{\partial t}$ $-\nabla \times \vec{E} = \frac{\partial \vec{B}}{\partial t} + \vec{J}_m$ $-\nabla \times \vec{E} = \frac{\partial \vec{B}}{\partial t} + \mu_0 \vec{J}_m$
    $\vec{F} = q_e (\vec{E} + \vec{v} \times \vec{B})$ $\vec{F} = q_e (\vec{E} + \vec{v} \times \vec{B}) + \frac{q_m}{\mu_0} (\vec{B} - \vec{v} \times \frac{\vec{E}}{c^2})$ $\vec{F} = q_e (\vec{E} + \vec{v} \times \vec{B}) + q_m (\vec{B} - \vec{v} \times \frac{\vec{E}}{c^2})$
  • 超材料：人工设计的具有特殊电磁特性（如负折射率）的材料，由John Pendry等学者研究，可实现负折射、完美透镜等特殊功能。

1.3 电磁场的能量和动量密度
#

1.3.1 能量密度与坡印廷矢量
#

  1. 能量密度$U$：电磁场单位体积内的能量，定义为： $$U = \frac{1}{2} (\vec{E} \cdot \vec{D} + \vec{B} \cdot \vec{H})$$
  2. 坡印廷矢量$\vec{S}$：描述电磁场的能量流动速率，即能流密度： $$\vec{S} = \vec{E} \times \vec{H}$$
  3. 能量守恒关系：无传导电流（$\vec{J}=0$）时，$\frac{\partial U}{\partial t} + \nabla \cdot \vec{S} = 0$，表明能量在体积内的变化率等于能流的净流入率。

1.3.2 色散介质中的有效能量密度
#

  • 海森堡不确定性原理：不存在单色波（$\Delta \omega \neq 0$），电场需用傅里叶变换表示：$\vec{D}(x,t) = \int_{-\infty}^{\infty} \tilde{D}(x,\omega) e^{-i\omega t} d\omega$。
  • 有效能量密度： $$u_{\text{eff}} = \frac{1}{2} \text{Re}\left[ \frac{d(\omega \varepsilon)}{d\omega}(\omega_0) |\tilde{D}(x,\omega_0)|^2 + \frac{d(\omega \mu)}{d\omega}(\omega_0) |\tilde{H}(x,\omega_0)|^2 \right]$$ 其中$\frac{d(\omega \varepsilon)}{d\omega}(\omega_0) = \varepsilon(\omega_0) + \omega_0 \frac{d\varepsilon}{d\omega}(\omega_0)$，体现色散对能量密度的影响。

1.3.3 电磁场的动量
#

  1. 动量密度$\vec{g}$：单位体积电磁场的动量，与坡印廷矢量的关系： $$\vec{g} = \frac{1}{c^2} \vec{S} = \mu_0 \varepsilon_0 \vec{E} \times \vec{H}$$
  2. 实例应用：
    • 光压：电磁波携带动量，照射物体时产生压力。
    • 光学牵引力：特殊结构下，光可对物体产生拉力（如$n_1 < n_2 < n_3$的介质结构）。

1.3.4 电磁场的质量
#

  • 由相对论质能方程$E = mc^2$，电磁场总能量$E = \iiint U d^3 r$，对应等效质量$m = \frac{E}{c^2}$，表明电磁场具有惯性质量。

1.4 色散与群速度
#

1.4.1 色散的分类与本质
#

  • 色散：电磁波的传播速度与频率相关的现象，本质是介质的介电常数$\varepsilon(\omega)$、磁导率$\mu(\omega)$随频率变化。
  • 分类：
    • 色度色散：频率导致的速度差异（本章核心）。
    • 空间色散：波矢$\vec{k}$导致的速度差异（1.5节）。

1.4.2 群速度与相速度
#

  1. 相速度$v_p$：单色波等相位面的传播速度，$v_p = \frac{\omega}{k}$。
  2. 群速度$v_g$：复色波包（信号）的传播速度，即能量传播速度，定义为： $$v_g = \left( \frac{d\omega}{dk} \right)_{\omega = \omega_0}$$
  3. 推导基础：
    • 两列近频平面波叠加：$U(z,t) = 2A \cos\left[ \frac{1}{2}(\Delta \omega t - \Delta k z) \right] e^{-i(\omega_0 t - k_0 z)}$，包络的传播速度为$\frac{\Delta \omega}{\Delta k}$，极限情况下为$v_g$。
    • 多列波叠加：通过傅里叶变换和$k(\omega)$的泰勒展开（$k(\omega) = k_0 + k’(\omega_0)(\omega - \omega_0) + \frac{1}{2}k’’(\omega_0)(\omega - \omega_0)^2$），得到包络$A(z,t) = A_0(t - k’(\omega_0) z)$，验证$v_g = \frac{1}{k’(\omega_0)}$。

1.4.3 群速度色散（GVD）
#

  • 定义：群速度随频率的变化，由$k(\omega)$的二阶导数描述：$D = -\frac{2\pi c}{\lambda^2} k’’(\omega_0)$。
  • 影响：导致脉冲展宽（如光纤中信号畸变），初始高斯脉冲$A(0,t) = e^{-at^2}$，传播后脉冲宽度随$z$增加。

1.4.4 群速度的特殊情况
#

  • $v_g < c$：正常色散介质（如常规光纤）。
  • $v_g > c$：反常色散介质（如某些超材料），不违背相对论（能量速度$v_e \leq c$）。
  • $v_g < 0$：负群速度，波包传播方向与相速度相反。
  • 能量速度与群速度关系：
    • 无耗介质：$v_g = v_e = \frac{c}{n_{\text{eff}}}$。
    • 弱耗介质：$v_g \approx v_e \leq \frac{c}{n_{\text{eff}}}$。
    • 强耗介质：$v_e \leq \frac{c}{n_{\text{eff}}}$，$v_g$无限制。

1.5 空间色散与光束扩展
#

1.5.1 空间色散的本质
#

  • 海森堡不确定性原理：不存在理想平面波（$\Delta k \neq 0$），光束的角谱分布导致传播时的空间色散。
  • 定义：介电常数$\varepsilon$与波矢$\vec{k}$相关（$\varepsilon = \varepsilon(\vec{k})$），导致光束传播特性依赖于空间分布。

1.5.2 光束的角谱表示
#

  • 傅里叶变换：任意光束可分解为不同方向的平面波叠加： $$U(x,y,0) = \int_{-\infty}^{\infty} A\left( \frac{\alpha}{\lambda}, \frac{\beta}{\lambda}, 0 \right) e^{j2\pi \left( \frac{\alpha}{\lambda}x + \frac{\beta}{\lambda}y \right)} d\frac{\alpha}{\lambda} d\frac{\beta}{\lambda}$$ 其中$\alpha, \beta$为方向余弦，传播后角谱项为$A\left( \frac{\alpha}{\lambda}, \frac{\beta}{\lambda}, z \right) = A\left( \frac{\alpha}{\lambda}, \frac{\beta}{\lambda}, 0 \right) e^{j\frac{2\pi}{\lambda} \sqrt{1 - \alpha^2 - \beta^2} z}$。

1.5.3 光束扩展的原因
#

  • 动量守恒：$k_z = \sqrt{k^2 - k_x^2 - k_y^2}$，旁轴近似下$k_z \approx k - \frac{k_x^2 + k_y^2}{2k}$，不同横向波矢$k_x, k_y$对应不同$k_z$，导致光束传播时展宽。
  • 空间色散系数：$D = -\frac{\partial^2 k_z}{\partial k_x^2} = \frac{1}{k}$，定量描述扩展程度。

1.5.4 显微镜分辨率的极限（阿贝极限）
#

  • 阿贝极限公式：$d = \frac{\lambda}{2n \sin\theta}$，其中$n$为介质折射率，$\theta$为物镜半孔径角。
  • 物理本质：空间色散导致光束无法聚焦到小于$\frac{\lambda}{2}$的光斑（空气介质中$n=1, \sin\theta=1$时，$d=\frac{\lambda}{2}$）。
  • 与不确定性原理的联系：$\Delta x \cdot \Delta p \geq \frac{\hbar}{2}$，$\Delta p \leq \frac{2h}{\lambda}$，推出$\Delta x \geq \frac{\lambda}{4\pi}$（近似为$\frac{\lambda}{2}$）。

1.5.5 特殊现象与应用
#

  • 古斯-汉欣效应：全反射时，光束在介质分界面发生横向位移，源于角谱成分的相位累积。
  • 慢光/快光：通过空间色散调控光束传播速度（如结构化光子的自由空间慢光，$v_g \approx c\left(1 - \frac{\langle k_x^2 \rangle}{2k_0^2}\right)$）。

1.6 费马原理、反射与折射
#

1.6.1 费马原理的形式
#

  1. 原始形式（最小时间原理）：光线在两点间传播的路径是耗时最短的路径。
    • 数学表达：$t = \frac{\sqrt{a^2 + x^2}}{v_1} + \frac{\sqrt{b^2 + (d - x)^2}}{v_2}$，对$x$求导并令$\frac{dt}{dx}=0$，得$\frac{\sin\theta_1}{v_1} = \frac{\sin\theta_2}{v_2}$。
  2. 现代形式（稳定光程原理）：光线传播路径的光程为极值（极小、极大或恒定），光程$S = \int_A^B n(\vec{r}) dl$，满足$\delta S = 0$（$\delta$为变分）。

1.6.2 斯涅尔定律（折射定律）
#

  • 推导：由费马原理，结合$v = \frac{c}{n}$，得$\frac{\sin\theta_1}{\sin\theta_2} = \frac{n_2}{n_1}$，即$n_1 \sin\theta_1 = n_2 \sin\theta_2$。
  • 反射定律：$\theta_1 = \theta_{\text{反射}}$，是斯涅尔定律在$n_1 = n_2$时的特例。

1.6.3 惠更斯原理
#

  • 核心：波前上的每个点都是次级球面波源，后续波前是这些次级波的包络。
  • 应用：解释反射、折射、衍射（如圆盘衍射的泊松亮斑）。

1.6.4 广义斯涅尔定律（超表面应用）
#

  • 超表面调控：通过超表面引入额外相位$\phi(x)$，修改光程$S = \int n dl + \phi(x)$，导致折射角满足$n_1 \sin\theta_1 - n_2 \sin\theta_2 = \frac{1}{k_0} \frac{d\phi}{dx}$。
  • 应用：实现负反射、负折射（波长1~1.9μm范围的超表面）。

1.7 电磁场的偏振
#

1.7.1 偏振的定义
#

  • 平面电磁波的偏振：电场矢量$\vec{E}$在垂直于传播方向的平面内的振动状态（磁场$\vec{B}$与$\vec{E}$垂直，偏振状态由$\vec{E}$决定）。
  • 平面波表达式：$\vec{E} = \vec{A} e^{i(\omega t - \vec{k} \cdot \vec{r})}$，满足$\vec{k} \cdot \vec{E} = 0$（横波特性）。

1.7.2 偏振的数学描述
#

设电场在$x,y$方向的分量为：

$$E_x = A_x \cos(\omega t + \delta_x), \quad E_y = A_y \cos(\omega t + \delta_y)$$

定义相位差$\delta = \delta_y - \delta_x$，根据$A_x, A_y$和$\delta$的不同，偏振态分为三类：

  1. 线偏振：
    • 条件：$\delta = 0$或$\pi$。
    • 振动方向：$\tan\theta = \frac{A_y}{A_x}$（$\delta=0$）或$\tan\theta = -\frac{A_y}{A_x}$（$\delta=\pi$）。
  2. 圆偏振：
    • 条件：$A_x = A_y = A$，$\delta = \pm \frac{\pi}{2}$。
    • 右旋圆偏振（RHC）：$\delta = -\frac{\pi}{2}$，电场矢量逆时针旋转。
    • 左旋圆偏振（LHC）：$\delta = \frac{\pi}{2}$，电场矢量顺时针旋转。
  3. 椭圆偏振：
    • 条件：任意$A_x, A_y$和$\delta$（非线偏振、圆偏振的情况）。
    • 椭圆方程：$\left( \frac{E_x}{A_x} \right)^2 + \left( \frac{E_y}{A_y} \right)^2 - 2 \frac{E_x E_y}{A_x A_y} \cos\delta = \sin^2\delta$。

1.7.3 偏振的表示方法
#

  1. 琼斯矢量（Jones Vector）：描述完全偏振光，常用形式：
    偏振态 琼斯矢量
    水平线偏振（H） $\begin{pmatrix} 1 \ 0 \end{pmatrix}$
    垂直线偏振（V） $\begin{pmatrix} 0 \ 1 \end{pmatrix}$
    右旋圆偏振（RHC） $\frac{1}{\sqrt{2}} \begin{pmatrix} 1 \ -i \end{pmatrix}$
    左旋圆偏振（LHC） $\frac{1}{\sqrt{2}} \begin{pmatrix} 1 \ i \end{pmatrix}$
    +45°线偏振 $\frac{1}{\sqrt{2}} \begin{pmatrix} 1 \ 1 \end{pmatrix}$
  2. 琼斯矩阵（Jones Matrix）：描述光学元件对偏振的调制，常用元件矩阵：
    光学元件 琼斯矩阵
    水平偏振器 $\begin{pmatrix} 1 & 0 \ 0 & 0 \end{pmatrix}$
    垂直偏振器 $\begin{pmatrix} 0 & 0 \ 0 & 1 \end{pmatrix}$
    快轴垂直的1/4波片 $e^{i\pi/4} \begin{pmatrix} 1 & 0 \ 0 & i \end{pmatrix}$
    快轴水平的1/4波片 $e^{i\pi/4} \begin{pmatrix} 1 & 0 \ 0 & -i \end{pmatrix}$
  3. 庞加莱球（Poincaré Sphere）：几何表示偏振态，球面上任意点对应一种完全偏振光，坐标$(S_1, S_2, S_3)$满足$S_1^2 + S_2^2 + S_3^2 = S_0^2$（$S_0$为光强）：
    • $S_0 = I$（光强）
    • $S_1 = I \cos2\psi$（$\psi$为振动方向角）
    • $S_2 = I \sin2\psi \cos\delta$（$\delta$为相位差）
    • $S_3 = I \sin2\psi \sin\delta$

1.7.4 布儒斯特角（Brewster Angle）
#

  • 定义：当反射光为完全线偏振光（垂直入射面）时的入射角$\theta_B$。
  • 公式：$\tan\theta_B = \frac{n_2}{n_1}$，此时反射光与折射光垂直（$\theta_B + \theta_2 = 90^\circ$）。

1.8 旋量与潘查拉特纳姆相位
#

1.8.1 旋量（Spinors）的数学描述
#

  • 定义：旋量是描述自旋为半整数粒子（如电子）和偏振光的数学工具，可视为“几何的平方根”。
  • 二维旋量形式：$\vec{s} = \sqrt{r} e^{-i\alpha/2} \begin{pmatrix} \cos(\theta/2) e^{-i\phi/2} \ \sin(\theta/2) e^{i\phi/2} \end{pmatrix}$，其中$r$为幅值，$\theta, \phi$为球坐标角，$\alpha$为相位。
  • 旋转特性：旋量绕$z$轴旋转$\theta_r$角后，形式为$\begin{pmatrix} e^{-i\theta_r/2} & 0 \ 0 & e^{i\theta_r/2} \end{pmatrix} \vec{s}$，需旋转$2\pi$才能恢复原旋量（角加倍效应）。

1.8.2 潘查拉特纳姆相位（Pancharatnam Phase）
#

  • 定义：偏振光在庞加莱球上沿闭合路径传播时，因几何效应获得的额外相位，属于几何相位。
  • 计算：相位值等于闭合路径包围的球面面积的一半（$\Delta\phi = \frac{1}{2} S$，$S$为路径包围的面积）。
  • 应用：超材料中的自旋选择透射、几何相位导光。

1.8.3 旋量与偏振的关联
#

  • 完全偏振光的旋量表示：旋量的模平方对应光强，相位对应偏振态的几何特性，与琼斯矢量、庞加莱球坐标满足一一对应关系（如右旋圆偏振对应旋量$\frac{1}{\sqrt{2}} \begin{pmatrix} 1 \ -i \end{pmatrix}$）。

1.9 平行输运及应用
#

1.9.1 列维-奇维塔平行输运（Levi-Civita Parallel Transport）
#

  • 定义：在弯曲空间（或曲面）中，矢量沿路径传播时保持“方向不变”的输运方式，需满足协变导数为零。
  • 平面上的平行输运：矢量方向不变，闭合路径后无旋转。
  • 球面上的平行输运：沿闭合路径（如球面三角形）输运后，矢量旋转角$\varphi = \int G d\Omega = \frac{S(\Omega)}{R^2}$（高斯-博内定理），其中$G$为高斯曲率，$S(\Omega)$为路径包围的球面面积。

1.9.2 应用实例
#

  1. 傅科摆（Foucault Pendulum）：地球自转导致摆平面旋转，本质是摆的振动矢量在地球表面（弯曲球面）的平行输运，旋转角$\alpha = 2\pi(1 - \sin\theta)$（$\theta$为纬度）。
  2. 电磁波的偏振旋转：
    • 法拉第旋转：线偏振光通过法拉第活性晶体时，偏振方向旋转$\theta = V B d$（$V$为费尔德常数，$B$为磁场强度，$d$为晶体厚度），源于磁场导致的能级分裂。
    • 几何相位偏振旋转：通过超材料的平行输运调控偏振方向。

1.9.3 高斯-博内-陈定理（Gauss-Bonnet-Chern Theorem）
#

  • 推广：将二维曲面的高斯-博内定理推广到高维黎曼流形，揭示几何曲率与拓扑性质的关联，是数学与物理（如规范场论）的重要桥梁。

1.10 原始麦克斯韦方程组及拓展
#

1.10.1 原始麦克斯韦方程组（麦克斯韦最初形式）
#

  • 包含矢量势$\vec{A}$和标量势$\Phi$，原始方程通过势函数描述电磁场： $$\vec{B} = \nabla \times \vec{A}, \quad \vec{E} = -\nabla\Phi - \frac{\partial \vec{A}}{\partial t}$$
  • 海维赛德-赫兹形式（简化形式）：即1.2.1中的方程组，省略势函数，突出$\vec{E}$和$\vec{H}$的对称性，但掩盖了势函数的物理意义。

1.10.2 势函数的物理意义
#

  • 传统观点：势函数是数学工具，无直接物理意义。
  • 现代观点：势函数是真实的物理存在，如阿哈罗诺夫-玻姆效应（AB效应）：
    • 现象：电子在$\vec{E} = \vec{B} = 0$的区域运动，但因矢量势$\vec{A}$存在，产生干涉条纹移动。
    • 相位差：$\Delta\phi = \frac{q}{\hbar c} \oint \vec{A} \cdot d\vec{l} = \frac{q \Phi_B}{\hbar c}$（$\Phi_B$为磁通量），证明势函数具有可观测的物理效应。

1.10.3 洛伦兹力与协变形式
#

  1. 洛伦兹力公式：$\vec{F} = q(\vec{E} + \vec{v} \times \vec{B})$，描述带电粒子在电磁场中的受力。
  2. 与科里奥利力的类比：科里奥利力$\vec{F}_c = 2m \vec{v} \times \vec{\omega}$（$\vec{\omega}$为旋转角速度），与洛伦兹力形式相似，均为速度相关的横向力，体现“力是几何效应的表象”。

1.10.4 电磁场的几何化
#

  • 外尔的尝试：试图将电磁场几何化，假设矢量传播时长度可变（规范变换），但被爱因斯坦质疑（尺子长度依赖历史）。
  • 量子力学的拯救：外尔的规范变换在量子力学中对应相位变换$\psi \to e^{-i\frac{q}{\hbar c} \int \vec{A} \cdot d\vec{l}} \psi$，与AB效应、潘查拉特纳姆相位一致，证明电磁场的几何本质。

1.11 电磁波的波粒二象性
#

1.11.1 波粒二象性的核心
#

  • 电磁波既具有波动性（干涉、衍射），又具有粒子性（光电效应、康普顿散射），光子是电磁场的量子化粒子。
  • 实验证据：
    • 波动性：双缝干涉（即使光子逐个入射，仍出现干涉条纹）。
    • 粒子性：光电效应中，光子能量$E = h\nu$，动量$p = \frac{h}{\lambda}$，与粒子碰撞时满足能量和动量守恒。

1.11.2 量子力学解释
#

  1. 哥本哈根诠释：微观粒子的状态由波函数描述，测量导致波函数坍缩，粒子从“叠加态”变为“确定态”（如薛定谔的猫思想实验）。
  2. 爱因斯坦的质疑：“光量子是什么？”，认为量子力学不完备，但后续实验（如贝尔不等式验证）支持量子力学的正确性。
  3. 玻尔的互补原理：波动性和粒子性是电磁波的互补属性，无法同时观测到（“深奥的真理，其对立面也是深奥的真理”）。

1.11.3 关键结论
#

  • 观测影响现实：电磁波的状态依赖于观测方式（如双缝实验中，观测缝则干涉条纹消失）。
  • 微观与宏观的区别：原子尺度的电磁波不遵循经典力学，需用量子力学描述（“原子不是石头或树那样的东西”）。

期末复习题
#

一、简答题
#

1. 写出麦克斯韦方程组的微分形式，并简要说明每个方程的物理意义。
答案：

  • $\nabla \cdot \vec{D} = \rho_f$：电场的散度与自由电荷密度成正比，描述电荷产生电场（高斯定律）。
  • $\nabla \cdot \vec{B} = 0$：磁场的散度为零，描述磁场是无源场（无磁单极子）。
  • $\nabla \times \vec{E} = -\frac{\partial \vec{B}}{\partial t}$：电场的旋度等于磁感应强度的时间变化率的负值，描述变化的磁场产生电场（法拉第电磁感应定律）。
  • $\nabla \times \vec{H} = \vec{J}_f + \frac{\partial \vec{D}}{\partial t}$：磁场的旋度由传导电流和位移电流共同产生，描述电流和变化的电场产生磁场（安培-麦克斯韦定律）。

2. 简述麦克斯韦引入位移电流的原因及位移电流的物理意义。
答案：

  • 引入原因：无位移电流时，$\nabla \cdot \vec{J}_f = 0$，结合电荷守恒$\nabla \cdot \vec{J}_f + \frac{\partial \rho_f}{\partial t} = 0$，会推出$\frac{\partial \rho_f}{\partial t} = 0$（无电荷积累），与电容器充电等实际现象矛盾，位移电流的引入解决了这一矛盾，保证电流连续性。
  • 物理意义：位移电流是电场变化率的体现（$\vec{J}_D = \frac{\partial \vec{D}}{\partial t}$），并非真实电荷流动，但能产生磁场，是电磁感应的逆过程，体现电场和磁场的对称性。

3. 什么是群速度和相速度？二者的关系是什么？群速度色散的物理意义是什么？
答案：

  • 相速度$v_p$：单色波等相位面的传播速度，$v_p = \frac{\omega}{k}$。
  • 群速度$v_g$：复色波包（信号）的传播速度，即能量传播速度，$v_g = \left( \frac{d\omega}{dk} \right)_{\omega_0}$。
  • 关系：$v_g = v_p - \lambda \frac{dv_p}{d\lambda}$（由$\omega = k v_p$，$k = \frac{2\pi}{\lambda}$推导），正常色散时$v_g < v_p$，反常色散时$v_g > v_p$。
  • 群速度色散（GVD）：群速度随频率的变化（由$k’’(\omega_0)$描述），物理意义是导致脉冲展宽（如光纤中信号畸变），因为不同频率成分的群速度不同，传播后脉冲宽度增加。

4. 简述费马原理的两种形式及斯涅尔定律的推导过程。
答案：

  • 费马原理形式：
    1. 原始形式（最小时间原理）：光线在两点间传播的路径是耗时最短的路径。
    2. 现代形式（稳定光程原理）：光线传播路径的光程为极值，$\delta S = \delta \int_A^B n dl = 0$。
  • 斯涅尔定律推导： 设两点$A、B$分别在折射率为$n_1、n_2$的介质中，光线在分界面的入射点横坐标为$x$，光程$S = n_1 \sqrt{a^2 + x^2} + n_2 \sqrt{b^2 + (d - x)^2}$，对$x$求导并令$\frac{dS}{dx}=0$，得$n_1 \frac{x}{\sqrt{a^2 + x^2}} = n_2 \frac{d - x}{\sqrt{b^2 + (d - x)^2}}$，结合$\sin\theta_1 = \frac{x}{\sqrt{a^2 + x^2}}$，$\sin\theta_2 = \frac{d - x}{\sqrt{b^2 + (d - x)^2}}$，推出$n_1 \sin\theta_1 = n_2 \sin\theta_2$（斯涅尔定律）。

5. 什么是阿贝极限？其物理本质是什么？如何突破阿贝极限？
答案：

  • 阿贝极限：显微镜的最小分辨距离，公式$d = \frac{\lambda}{2n \sin\theta}$，其中$n$为介质折射率，$\theta$为物镜半孔径角。
  • 物理本质：空间色散导致光束无法聚焦到小于$\frac{\lambda}{2}$的光斑，源于海森堡不确定性原理（$\Delta x \cdot \Delta p \geq \frac{\hbar}{2}$），横向位置不确定度受限于光的波长。
  • 突破方法：使用超材料（如负折射率材料实现完美透镜）、近场光学显微镜、受激辐射耗尽显微镜（STED）等。

6. 简述电磁波偏振的三种类型及判断条件。
答案： 设电场分量$E_x = A_x \cos(\omega t + \delta_x)$，$E_y = A_y \cos(\omega t + \delta_y)$，相位差$\delta = \delta_y - \delta_x$：

  • 线偏振：$\delta = 0$或$\pi$，电场矢量沿固定方向振动，$\tan\theta = \pm \frac{A_y}{A_x}$。
  • 圆偏振：$A_x = A_y$且$\delta = \pm \frac{\pi}{2}$，$\delta = -\frac{\pi}{2}$为右旋圆偏振（逆时针旋转），$\delta = \frac{\pi}{2}$为左旋圆偏振（顺时针旋转）。
  • 椭圆偏振：任意$A_x、A_y$和$\delta$（非线偏振、圆偏振条件），电场矢量轨迹为椭圆，满足椭圆方程$\left( \frac{E_x}{A_x} \right)^2 + \left( \frac{E_y}{A_y} \right)^2 - 2 \frac{E_x E_y}{A_x A_y} \cos\delta = \sin^2\delta$。

7. 什么是阿哈罗诺夫-玻姆效应？其物理意义是什么？
答案：

  • 现象：电子在$\vec{E} = \vec{B} = 0$的区域运动，但周围存在磁场（矢量势$\vec{A} \neq 0$），电子的干涉条纹会发生移动。
  • 物理意义：证明电磁场的势函数（$\vec{A}$、$\Phi$）是真实的物理存在，而非单纯的数学工具；电磁场的物理效应不仅由$\vec{E}$和$\vec{B}$决定，还与势函数相关，支持电磁场的几何化描述。

8. 简述电磁波的波粒二象性及实验证据。
答案：

  • 波粒二象性：电磁波既具有波动性（干涉、衍射），又具有粒子性（光电效应、康普顿散射），光子是其量子化粒子，能量$E = h\nu$，动量$p = \frac{h}{\lambda}$。
  • 实验证据：
    • 波动性：双缝干涉实验（即使光子逐个入射，仍出现明暗相间的干涉条纹）、单缝衍射实验。
    • 粒子性：光电效应（光子撞击金属表面，打出光电子，能量满足$E_k = h\nu - W_0$，$W_0$为逸出功）、康普顿散射（光子与自由电子碰撞，波长变长，满足动量和能量守恒）。

二、计算题
#

1. 已知真空中某单色光的波长$\lambda = 500 , \text{nm}$，求其频率$\nu$、角频率$\omega$和波矢$k$的大小。（真空中光速$c = 3 \times 10^8 , \text{m/s}$） 答案：

  • 频率：$\nu = \frac{c}{\lambda} = \frac{3 \times 10^8}{500 \times 10^{-9}} = 6 \times 10^{14} , \text{Hz}$。
  • 角频率：$\omega = 2\pi\nu = 2\pi \times 6 \times 10^{14} = 3.77 \times 10^{15} , \text{rad/s}$。
  • 波矢大小：$k = \frac{2\pi}{\lambda} = \frac{2\pi}{500 \times 10^{-9}} = 1.26 \times 10^7 , \text{rad/m}$。

2. 一束光从空气（$n_1 = 1$）入射到玻璃（$n_2 = 1.5$）表面，入射角$\theta_1 = 30^\circ$，求折射角$\theta_2$；若入射角为布儒斯特角$\theta_B$，求$\theta_B$的大小。 答案：

  • 由斯涅尔定律$n_1 \sin\theta_1 = n_2 \sin\theta_2$，得： $$\sin\theta_2 = \frac{n_1}{n_2} \sin\theta_1 = \frac{1}{1.5} \times \sin30^\circ = \frac{1}{1.5} \times 0.5 = \frac{1}{3} \approx 0.333$$ $$\theta_2 = \arcsin(0.333) \approx 19.47^\circ$$
  • 布儒斯特角公式$\tan\theta_B = \frac{n_2}{n_1}$，得： $$\theta_B = \arctan\left( \frac{1.5}{1} \right) = \arctan(1.5) \approx 56.31^\circ$$

3. 已知某介质中电磁波的色散关系为$\omega = k c / n$（$n$为常数，无色散），求相速度$v_p$和群速度$v_g$，并说明该介质的色散特性。 答案：

  • 相速度：$v_p = \frac{\omega}{k} = \frac{c}{n}$。
  • 群速度：$v_g = \frac{d\omega}{dk} = \frac{c}{n}$。
  • 色散特性：$v_p = v_g$，群速度色散$D = -\frac{2\pi c}{\lambda^2} k’’(\omega) = 0$，该介质为无色散介质，脉冲传播时不会展宽。

4. 一束线偏振光的琼斯矢量为$\begin{pmatrix} 1 \ 1 \end{pmatrix}$，通过快轴垂直的1/4波片（琼斯矩阵为$\begin{pmatrix} 1 & 0 \ 0 & i \end{pmatrix}$）后，求出射光的琼斯矢量，并判断其偏振态。 答案：

  • 出射琼斯矢量：$\vec{J}_{\text{出}} = \begin{pmatrix} 1 & 0 \ 0 & i \end{pmatrix} \begin{pmatrix} 1 \ 1 \end{pmatrix} = \begin{pmatrix} 1 \ i \end{pmatrix}$。
  • 偏振态判断：将矢量归一化，$\frac{1}{\sqrt{2}} \begin{pmatrix} 1 \ i \end{pmatrix}$，对应左旋圆偏振光（因$E_y$分量相位比$E_x$超前$\frac{\pi}{2}$）。

第二章 光学谐振腔与粒子散射
#

2.1 光学谐振腔简介
#

2.1.1 谐振腔的基本条件
#

  • 玻尔量子化条件：谐振腔中传播的电磁波需满足相位匹配，即往返相位差为$2m\pi$（$m$为整数，量子数），数学表达为$\delta\phi = 2m\pi$。只有满足该条件的电磁波才能在腔内稳定存在，形成谐振模式。
  • 拓扑特性：谐振腔的几何结构遵循欧拉示性数$V-E+F=2$（$V$为顶点数、$E$为棱数、$F$为面数），常见规则多面体（四面体、立方体、八面体等）均满足该关系，保证腔内电磁波的闭合传播路径。

2.1.2 谐振腔的关键参数
#

  1. 品质因数（Q值）：

    • 定义：$Q = \omega \times \frac{\text{谐振腔存储的场能量}}{\text{谐振腔损耗的功率}}$，反映谐振腔存储能量与损耗的比值。
    • 能量存储：电场能量$U_{electric} = \frac{1}{8}\varepsilon E_0^2 AL$，磁场能量$U_{magnetic} = U_{electric}$，总存储能量$U = \frac{1}{4}\varepsilon E_0^2 V$（$V$为腔体积）。
    • 物理意义：Q值越高，腔的损耗越小，谐振峰越尖锐，单色性越好（如激光腔的Q值可达$10^6$以上）。

  2. 模式密度：

    • 波矢关系：腔内电磁波满足$k_x^2 + k_y^2 + k_z^2 = \left(\frac{\omega n}{c}\right)^2$（$n$为介质折射率），谐振条件为$k_x = \frac{r\pi}{a}$、$k_y = \frac{s\pi}{b}$、$k_z = \frac{t\pi}{c}$（$r,s,t$为整数，$a,b,c$为腔的三维尺寸）。
    • 模式密度公式：单位频率间隔内的模式数$p(\nu) = \frac{8\pi\nu^2 n^3 V}{c^3}$（无色散时）；考虑色散时$p(\nu) = \frac{3\pi\nu^2 n^2 V\left(n + \nu \frac{\partial n}{\partial \nu}\right)}{c^3}$。

2.1.3 波导与谐振腔的关联
#

  • 谐振腔可视为波导的“闭合版”：波导通过边界约束电磁波沿一维传播，谐振腔则通过两端反射镜（或闭合边界）使电磁波往返传播，形成驻波模式（如TE模、TM模）。
  • 模式演化：随着腔尺寸与波长比值（$t/\lambda$）增大，模式数量增多（TE0、TE1、TM0、TM1等），有效折射率$n_{eff}$呈现离散分布。

补充光学例子：

  • 激光谐振腔：He-Ne激光器的谐振腔由两个反射镜组成（一个全反射镜，一个部分反射镜），满足玻尔量子化条件的红光（$\lambda=632.8nm$）在腔内谐振，输出单色性极佳的激光，Q值越高，激光线宽越窄。
  • 微波炉谐振腔：微波炉的金属腔体构成谐振腔，工作频率约2.45GHz，对应的腔尺寸与波长（$\lambda≈12cm$）匹配，使微波在腔内形成驻波，集中能量加热食物，体现模式约束与能量存储特性。

2.2 法布里-珀罗谐振腔（F-P腔）
#

2.2.1 结构与工作原理
#

  • 基本结构：由两块平行反射镜组成，中间填充折射率为$n$的介质（气体、固体或液体），镜间距为$l$，反射率分别为$R_1$和$R_2$。
  • 相位差与光程差：入射光以角度$\theta$入射时，往返光程差$\Delta L = 2l\cos\theta$，相位差$\delta = \frac{4\pi n l \cos\theta}{\lambda}$（$\lambda$为真空中波长）。

2.2.2 透射与反射特性
#

  1. 透射率公式： $$\frac{I_t}{I_i} = \frac{(1-R)^2}{(1-R)^2 + 4R\sin^2(\delta/2)}$$ 其中$R = r^2$（$r$为振幅反射系数），$T = 1-R$为单程透射率。
  2. 谐振条件：当$\delta = 2m\pi$（$m$为整数）时，透射率达到最大值1（相干相长干涉），对应的谐振频率$\nu_m = m\frac{c}{2nl\cos\theta}$。
  3. 自由光谱范围（FSR）：相邻两个谐振峰的频率间隔$\Delta\nu = \frac{c}{2nl\cos\theta}$，反映腔的频率分辨能力。

2.2.3 腔内场与群延迟
#

  • 腔内光强：谐振时腔内平均光强$I_{ave} = \frac{1+R}{1-R}I_i$，当$R=0.99$时，$I_{ave}=199I_i$，体现腔内能量增强效应。
  • 群延迟增强：透射光的群延迟$\tau_g = \frac{1+R}{1-R}\cdot\frac{2nl\cos\theta}{c}$，高反射率下群延迟显著增大，可用于慢光器件。

补充光学例子：

  • 光谱分析仪：F-P腔作为核心部件，利用其高分辨率（透射峰窄）特性，可分辨原子光谱的精细结构（如氢原子的巴尔末线系），自由光谱范围决定了可测量的频率范围，分辨率由Q值决定。
  • LIGO引力波探测器：LIGO的核心是长达4km的F-P腔，通过激光在腔内往返传播（约50次），将引力波引起的微小长度变化（$<10^{-18}m$）放大，利用干涉条纹变化探测引力波，体现F-P腔的光程放大与高灵敏度特性。
  • 肥皂泡的彩色条纹：肥皂泡的前后表面构成微型F-P腔，不同波长的光满足不同的谐振条件，呈现彩色干涉条纹，当肥皂泡变薄时，光程差减小，条纹颜色随厚度变化，最终破裂前因厚度趋近于零，呈现黑色（相消干涉）。

2.3 光学谐振腔的损耗
#

2.3.1 损耗机制
#

  1. 非理想反射损耗：反射镜反射率$R<1$，部分光透射或吸收，导致能量损耗，是腔的主要损耗来源之一。
  2. 吸收与散射损耗：腔内介质（如增益介质、空气）的吸收，以及镜面、介质中的杂质散射，使光能量转化为热能或向非指定方向传播。
  3. 衍射损耗：当腔尺寸与波长可比时，电磁波会发生衍射，超出腔的约束范围，造成能量泄漏（如微腔的衍射损耗显著）。

2.3.2 腔的寿命与线宽
#

  1. 模式寿命$\tau_c$：腔内能量衰减到初始值$1/e$的时间，公式为： $$\tau_c = \frac{n}{c\left[\alpha - \frac{1}{l}\ln\sqrt{R_1R_2}\right]}$$ 其中$\alpha$为介质的吸收系数，$R_1,R_2$为反射镜反射率。
  2. 谐振线宽$\Delta\nu_{1/2}$：与寿命满足$\Delta\nu_{1/2} = \frac{1}{2\pi\tau_c}$，Q值与线宽的关系为$Q = \frac{\nu_0}{\Delta\nu_{1/2}}$（$\nu_0$为谐振频率），Q值越高，线宽越窄。

2.3.3 相干完美吸收（CPA）
#

  • 定义：当两束相干光入射到吸收腔时，通过相消干涉使入射光完全被吸收（$T=0$，$R=0$），能量被腔完全吸收，无反射和透射。
  • 条件：$\sqrt{R_1} = A$（$A = e^{-\alpha l}$为介质单程透射率），核心机制是相干相消干涉，而非单纯的介质吸收，体现损耗与干涉的协同作用。

补充光学例子：

  • 低损耗激光腔：He-Ne激光器的反射镜反射率$R≈0.999$，介质吸收系数$\alpha$极小，模式寿命$\tau_c$可达毫秒级，Q值高，输出激光线宽仅约1MHz，单色性极佳。
  • 抗激光（Anti-laser）：基于相干完美吸收原理，将激光的“受激辐射”逆转为“受激吸收”，入射激光在腔内完全吸收，可用于激光防护（如吸收特定波长的激光，保护探测器或人眼）。
  • 微腔的衍射损耗：直径为几微米的微球谐振腔，因尺寸小，衍射损耗显著，Q值相对较低（约$10^3$），但体积小、集成度高，可用于片上光子器件（如微腔激光器）。

2.4 多腔谐振腔
#

2.4.1 多层膜与增透膜
#

  • 结构：由多层不同折射率的介质膜组成（如两层膜$n_1-n_2-n_3$），等效于多个串联的F-P腔，利用多光束干涉实现特定光学功能。
  • 增透膜条件：当$n_2 = \sqrt{n_1n_3}$且光程差$\phi = \pi/2, 3\pi/2,…$时，反射率$R=0$，实现增透（如照相机镜头的增透膜，减少反射光，提高透光率）。

2.4.2 布拉格反射器（Bragg Grating）
#

  • 结构：由交替排列的高折射率（$n_2$）和低折射率（$n_1$）介质层组成，每层厚度为$\lambda/4$（中心波长），等效于周期性多腔结构。
  • 反射特性：当入射光波长满足布拉格条件$2n_d d = m\lambda$（$n_d$为平均折射率，$d$为周期厚度，$m$为整数）时，反射率接近100%，形成布拉格禁带（特定波长范围的光被强烈反射）。

2.4.3 光子晶体与安德森局域化
#

  1. 光子晶体：折射率周期性排列的人工结构（如二维光子晶体光纤），其布拉格禁带可调控光的传播，实现光的局域化或无损耗传输，是多腔谐振腔的延伸。
  2. 安德森局域化：在无序介质（如随机分布的散射颗粒）中，光因多重散射被局域在有限空间内，无法传播，形成随机激光（增益介质+无序散射体，无需传统反射镜）。

补充光学例子：

  • 光纤布拉格光栅（FBG）：在光纤纤芯中写入周期性折射率调制结构（布拉格反射器），反射特定波长的光（如1550nm通信波长），用于光纤通信中的波长选择、色散补偿，或作为传感器（通过反射波长偏移测量温度、应变）。
  • 增透膜的应用：眼镜镜片表面的增透膜（通常为$MgF_2$，折射率$n≈1.38$），满足$n_2 = \sqrt{n_{空气}n_{玻璃}}$（$n_{空气}=1$，$n_{玻璃}=1.5$），使可见光反射率从约4%降至1%以下，减少眩光，提高视觉清晰度。
  • 随机激光：在染料溶液中混入纳米颗粒（无序散射体），泵浦光激发染料产生增益，光经多重散射局域化，形成激光输出，无需传统谐振腔，可用于柔性显示、生物传感等场景。

2.5 粒子散射基础
#

2.5.1 散射的基本概念
#

  • 定义：电磁波入射到粒子（如原子、分子、纳米颗粒）时，因粒子与电磁场的相互作用，光的传播方向改变（散射）、能量被吸收（吸收），或穿透粒子（透射），总效应为消光（消光=散射+吸收）。
  • 边界条件：粒子表面满足电场和磁场的切向分量连续（$\vec{E}_1^t = \vec{E}_2^t$，$\vec{H}_1^t = \vec{H}_2^t$），法向分量满足$\vec{D}_1^n = \vec{D}_2^n$，$\vec{B}_1^n = \vec{B}_2^n$。

2.5.2 关键物理量
#

  1. 截面：
    • 消光截面$C_{ext}$：$\frac{W_{ext}}{I_i}$（$W_{ext}$为消光功率，$I_i$为入射光强），描述粒子对光的阻挡能力。
    • 散射截面$C_{sca}$：$\frac{W_{sca}}{I_i}$，描述散射光的总功率。
    • 吸收截面$C_{abs}$：$\frac{W_{abs}}{I_i}$，描述粒子吸收的光功率，满足$C_{ext} = C_{sca} + C_{abs}$。
  2. 效率因子：$Q_{ext} = \frac{C_{ext}}{A}$、$Q_{sca} = \frac{C_{sca}}{A}$、$Q_{abs} = \frac{C_{abs}}{A}$（$A$为粒子的几何截面），反映粒子的散射/吸收能力相对于几何尺寸的比值。

2.5.3 光学定理
#

  • 核心内容：消光截面仅由前向散射振幅决定，公式为： $$C_{ext} = \frac{4\pi}{k} \text{Re}[f(0)]$$ 其中$k = \frac{2\pi}{\lambda}$，$f(0)$为前向（$\theta=0$）散射振幅。
  • 物理意义：即使粒子对光有吸收，消光效应也可通过前向散射的干涉现象体现，揭示了散射与吸收的内在关联。

补充光学例子：

  • 雾霾天气的能见度下降：雾霾中的颗粒物（直径$0.1-1\mu m$）对可见光的散射截面大，$Q_{sca}$高，大量散射使光线传播方向杂乱，能见度降低，体现粒子散射对光传播的影响。
  • 太阳镜的吸收作用：太阳镜镜片中的金属离子（如铜离子、铁离子）对紫外线和部分可见光有吸收截面$C_{abs}$，吸收有害光线，保护眼睛，同时镜片表面的偏振膜利用散射原理过滤水平偏振光（眩光）。
  • 激光粒度仪：利用粒子散射光的角分布（与粒子尺寸相关），通过测量散射光强随角度的变化，反推粒子的尺寸分布，核心依据是散射截面与粒子尺寸的依赖关系。

2.6 米氏理论：球形粒子散射
#

2.6.1 米氏理论的适用范围
#

  • 核心：描述任意尺寸的球形粒子对电磁波的散射，当粒子尺寸参数$x = \frac{2\pi a}{\lambda}$（$a$为粒子半径）满足任意值（$x\ll1$为瑞利散射，$x≈1$为米氏散射，$x\gg1$为几何散射）。
  • 特殊情况：瑞利散射（$x\ll1$，如空气分子、纳米颗粒），米氏散射（$x≈1$，如云层中的小水滴、花粉），几何散射（$x\gg1$，如雨滴、玻璃球）。

2.6.2 场的展开与散射矩阵
#

  1. 场的表达式：入射光、散射光和粒子内部的场均用球谐函数和球汉克尔函数展开，满足边界条件后，得到散射振幅系数$a_n$和$b_n$（$n$为多极子阶数，$n=1$为偶极子，$n=2$为四极子，$n=3$为六极子）。
  2. 散射矩阵：描述入射光偏振态与散射光偏振态的关系，形式为： $$\begin{pmatrix} E_{\parallel s} \\ E_{\perp s} \end{pmatrix} = \frac{e^{ik(r - z)}}{-ikr} \begin{pmatrix} S_2 & 0 \\ 0 & S_1 \end{pmatrix} \begin{pmatrix} E_{\parallel i} \\ E_{\perp i} \end{pmatrix}$$ 其中$S_1 = \sum_{n=1}^{\infty} \frac{2n+1}{n(n+1)}(a_n u_n + b_n \tau_n)$，$S_2 = \sum_{n=1}^{\infty} \frac{2n+1}{n(n+1)}(a_n \tau_n + b_n u_n)$（$u_n$、$\tau_n$为角相关函数）。

2.6.3 关键结论与特殊情况
#

  1. 瑞利散射（$x\ll1$）：
    • 散射振幅系数：$a_1 ≈ -\frac{2x^3}{3}\frac{m^2 - 1}{m^2 + 2}$，$b_1 ≈ -\frac{x^3}{45}(m^2 - 1)$（$m = \frac{n_{粒子}}{n_{介质}}$为相对折射率）。
    • 散射光强：$I_s ∝ \frac{1}{\lambda^4}$（瑞利散射定律），无偏振入射时$I_s = \frac{8\pi^4 N a^6}{\lambda^4 r^2} \left|\frac{m^2 - 1}{m^2 + 2}\right|^2 (1 + \cos^2\theta) I_i$（$N$为粒子数密度）。
  2. 米氏散射（$x≈1$）：
    • 散射光强与$\lambda$无明显依赖关系（$I_s ∝ \lambda^0$），散射图案呈前后不对称分布，前向散射强于后向散射。
  3. 多极子贡献：小粒子（$x\ll1$）以偶极子散射为主，大粒子（$x\gg1$）需考虑四极子、六极子等高阶多极子散射。

补充光学例子：

  • 天空的蓝色与日出的红色：空气分子（$a≈0.1nm$，$x\ll1$）对可见光的瑞利散射，蓝光（$\lambda≈450nm$）散射强度是红光（$\lambda≈650nm$）的$(\frac{650}{450})^4≈6$倍，故天空呈蓝色；日出/日落时，太阳光穿过更长路径的大气层，蓝光被大量散射，剩余红光主导，呈现红色。
  • 云的白色：云层中的小水滴（$a≈10\mu m$，$x≈10$）属于米氏散射，散射光强与波长无关，所有颜色的光被均匀散射，故云呈白色；乌云因水滴浓度高，光线被多次散射和吸收，透射光减少，呈现黑色。
  • 金属纳米颗粒的表面等离激元共振：银纳米颗粒（$a≈50nm$）在可见光波段（$\lambda≈400-500nm$）发生偶极子共振，散射截面$C_{sca}$显著增大，呈现强烈的黄色，可用于生物标记（通过散射光识别纳米颗粒的位置）或表面增强拉曼散射（SERS）基底。
  • 等离子体癌症治疗：将金纳米壳（硅芯+金壳，$a≈100nm$）注入肿瘤组织，金纳米壳在近红外激光（$\lambda≈800nm$）照射下发生等离激元共振，强烈吸收光能量并转化为热能，加热杀死肿瘤细胞，而健康组织不受影响，体现米氏散射中吸收与共振的协同作用。

2.7 金属粒子谐振的几何化
#

2.7.1 金属的介电特性
#

  • 金属的相对介电常数$\varepsilon_m$具有负的实部（$\text{Re}[\varepsilon_m] < 0$），源于自由电子的集体振荡（等离子体振荡），如银在可见光波段$\text{Re}[\varepsilon_m]≈-10$，$\text{Im}[\varepsilon_m]$较小（低损耗）。
  • 表面等离激元（SPP）：金属与介质界面处的自由电子振荡与电磁波耦合形成的表面模式，波矢$k_{spp} = \frac{2\pi}{\lambda}\sqrt{\frac{\varepsilon_m \varepsilon_d}{\varepsilon_m + \varepsilon_d}}$（$\varepsilon_d$为介质介电常数），具有局域场增强效应。

2.7.2 玻尔条件的几何诠释
#

  • 金属粒子的谐振满足几何化的玻尔条件：$\oint k(r) dr = 2m\pi$（$m$为整数），其中$k(r)$为粒子内的有效波矢，与表面等离激元的传播路径相关。
  • 谐振模式：$m=1$为偶极子模式，$m=2$为四极子模式，$m=3$为六极子模式，高阶模式的谐振频率高于低阶模式（蓝移），且Q值更高（损耗更小）。

2.7.3 谐振频率的尺寸依赖
#

  • 尺寸效应：金属粒子半径$a$减小，谐振频率蓝移（如银纳米颗粒，$a$从100nm减小到50nm，谐振波长从500nm蓝移到400nm）；半径增大，谐振频率红移，源于表面等离激元的传播路径长度变化。
  • 模式重叠与平坦带：当粒子结构对称时，不同阶数的模式可能发生重叠（平坦带，$d\omega/dk=0$），导致散射图案重构；反向模式（$d\omega/dk<0$）则会产生异常散射（如后向散射增强）。

补充光学例子：

  • 彩色金属纳米颗粒溶液：不同尺寸的金纳米颗粒溶液呈现不同颜色，$a=10nm$时呈红色，$a=50nm$时呈紫色，$a=100nm$时呈蓝色，源于谐振频率的尺寸依赖，可用于彩色显示、防伪标记。
  • 表面增强拉曼散射（SERS）：金纳米颗粒聚集时，颗粒间隙处的表面等离激元共振使局域电场增强$10^4-10^6$倍，吸附在间隙处的分子的拉曼散射信号显著增强，可用于单分子检测（如生物分子、环境污染物），核心是金属粒子谐振的局域场增强效应。
  • 等离激元波导：由金属纳米线（如银纳米线，$a=50nm$）构成的波导，表面等离激元沿纳米线传播，模式尺寸小于衍射极限（$\lambda/10$），可用于片上高密度光互连，突破传统光纤的衍射极限限制。

2.8 弛豫全反射（Relaxed TIR）
#

2.8.1 基本原理
#

  • 传统全反射（TIR）：当光从光密介质（$n_1$）入射到光疏介质（$n_2$），且入射角$\theta > \theta_c = \arcsin(n_2/n_1)$时，发生全反射，光场在光疏介质中形成倏逝波（穿透深度$\approx\lambda$）。
  • 弛豫全反射：通过引入各向异性介质或亚波长结构，放松全反射的条件（$\theta \leq \theta_c$也可实现强局域），使光场局域在介质界面，局域尺寸突破衍射极限（$\lambda/10$以下），且无金属损耗（与等离子体模式相比）。

2.8.2 应用场景
#

  • 亚波长光 confinement：用于片上光学器件（如波导、调制器），实现光的纳米尺度约束，提高器件集成度。
  • 透明亚衍射光学：无需金属即可实现光的亚波长局域，避免金属损耗，适用于高传输效率的光互连、生物成像（如超分辨显微镜）。

补充光学例子：

  • 超分辨显微镜：基于弛豫全反射原理的透明亚衍射透镜，可将光聚焦到$\lambda/20$的光斑，突破阿贝极限，用于观察细胞内的纳米结构（如细胞器、生物大分子），无需荧光标记，成像分辨率高且对生物样本无损伤。
  • 片上亚波长波导：由各向异性介质构成的弛豫全反射波导，芯层尺寸仅50nm（$\lambda=1550nm$时，$\lambda/30$），光场局域在芯层与包层界面，传输损耗低（<0.1dB/cm），可用于高密度光芯片（如光子计算机的互连线路）。

期末复习题（简答+计算）
#

一、简答题
#

1. 简述光学谐振腔的玻尔量子化条件及物理意义。
答案：

  • 条件：谐振腔内往返传播的电磁波需满足相位匹配，即$\delta\phi = 2m\pi$（$m$为整数，量子数），对应的光程差$\Delta L = m\lambda/2n$（$n$为介质折射率，$\lambda$为真空中波长）。
  • 物理意义：只有满足该条件的电磁波才能在腔内形成稳定驻波（谐振模式），不满足条件的电磁波因相消干涉被抑制；量子数$m$对应模式的阶数，决定谐振频率的大小。

2. 什么是谐振腔的品质因数（Q值）？其物理意义及影响因素是什么？
答案：

  • 定义：$Q = \omega \times \frac{\text{腔内存储的场能量}}{\text{单位时间内损耗的能量}}$，是描述谐振腔损耗与频率选择性的核心参数。
  • 物理意义：Q值越高，腔内能量损耗越小，谐振峰越尖锐（线宽越窄），频率选择性越好（如激光腔的高Q值保证输出激光的单色性）。
  • 影响因素：
    1. 反射损耗：反射镜反射率$R$越低，损耗越大，Q值越小。
    2. 吸收与散射损耗：腔内介质、镜面的吸收和散射，导致能量转化或泄漏，降低Q值。
    3. 衍射损耗：腔尺寸越小（与波长可比），衍射损耗越大，Q值越小。

3. 简述法布里-珀罗谐振腔的透射特性及谐振条件。
答案：

  • 透射特性：透射率随入射光频率（或相位差）周期性变化，形成尖锐的透射峰（谐振峰）和谷（相消干涉），透射率公式为$\frac{I_t}{I_i} = \frac{(1-R)^2}{(1-R)^2 + 4R\sin^2(\delta/2)}$（$R$为反射率，$\delta$为往返相位差）。
  • 谐振条件：当往返相位差$\delta = 2m\pi$（$m$为整数）时，透射率达到最大值1（相干相长干涉），对应的谐振频率$\nu_m = m\frac{c}{2nl\cos\theta}$（$l$为镜间距，$\theta$为入射角度）。
  • 自由光谱范围：相邻谐振峰的频率间隔$\Delta\nu = \frac{c}{2nl\cos\theta}$，反映腔的频率分辨范围。

4. 区分瑞利散射与米氏散射的适用条件、散射特性及对应的物理现象。 答案：

特性 瑞利散射 米氏散射
适用条件 粒子尺寸参数$x = 2\pi a/\lambda \ll 1$（如空气分子、纳米颗粒） 粒子尺寸参数$x ≈ 1$（如云层小水滴、花粉）
散射光强 $I_s ∝ 1/\lambda^4$（强烈依赖波长） $I_s ∝ \lambda^0$（与波长无关）
散射图案 前后对称（偶极子散射） 前向散射强于后向散射（不对称）
物理现象 天空蓝色、日出红色、晚霞 云的白色、雾霾能见度下降

5. 简述光学定理的内容及物理意义。
答案：

  • 内容：消光截面仅由前向散射振幅决定，公式为$C_{ext} = \frac{4\pi}{k} \text{Re}[f(0)]$（$k=2\pi/\lambda$，$f(0)$为前向散射振幅）。
  • 物理意义：消光是散射与吸收的总和，但消光截面可通过测量前向散射单独获得，无需测量所有方向的散射光；揭示了散射与吸收的内在关联，即使粒子有吸收，也可通过前向散射的干涉效应体现消光效应。

6. 什么是表面等离激元共振？简述金属纳米颗粒的表面等离激元共振特性及应用。 答案：

  • 定义：金属表面的自由电子在电磁波照射下发生集体振荡，与入射光耦合形成的共振现象，是金属粒子谐振的核心机制。
  • 特性：共振时散射截面$C_{sca}$和吸收截面$C_{abs}$显著增大，局域电场增强，共振频率依赖于粒子尺寸（尺寸减小，共振频率蓝移）、形状和周围介质折射率。
  • 应用：生物标记（通过共振散射识别纳米颗粒位置）、表面增强拉曼散射（SERS）基底（局域场增强拉曼信号）、等离子体癌症治疗（共振吸收转化为热能杀死肿瘤细胞）、彩色显示（不同尺寸颗粒呈现不同颜色）。

7. 简述多腔谐振腔中布拉格反射器的工作原理及应用。 答案：

  • 工作原理：由交替排列的高折射率和低折射率介质层组成，每层厚度为$\lambda/4$（中心波长），当入射光满足布拉格条件$2n_d d = m\lambda$（$n_d$为平均折射率，$d$为周期厚度，$m$为整数）时，各层反射光相干相长，反射率接近100%，形成布拉格禁带。
  • 应用：光纤布拉格光栅（FBG，用于波长选择、色散补偿、传感器）、分布式反馈激光器（DFB-LD，通过布拉格反射器实现波长锁定）、增透膜/高反膜（多层膜结构，实现特定波长的增透或高反）。
