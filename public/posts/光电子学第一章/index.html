第一章 电磁理论
#

1.1 电磁相互作用简介
#

核心内容
#

  • 自然界四大基本相互作用包括引力、电磁相互作用、弱核力和强核力。
  • 电磁相互作用广泛存在于微观与宏观世界，涉及电子、光子等粒子的相互作用过程。

1.2 麦克斯韦方程组与电磁统一
#

麦克斯韦方程组的积分形式与微分形式
#

名称 积分方程 微分方程
高斯定律 $\oiint_{\partial \Omega} D \cdot dS = \iiint_{\Omega} \rho_f dV$ $\nabla \cdot D = \rho_f$
磁高斯定律 $\oiint_{\partial \Omega} B \cdot dS = 0$ $\nabla \cdot B = 0$
法拉第电磁感应定律 $\oint_{\partial \Sigma} E \cdot dl = -\frac{d}{dt} \iint_{\Sigma} B \cdot dS$ $\nabla \times E = -\frac{\partial B}{\partial t}$
安培环路定律（含麦克斯韦修正） $\oint_{\partial \Sigma} H \cdot dl = \iint_{\Sigma} J_f \cdot dS + \frac{d}{dt} \iint_{\Sigma} D \cdot dS$ $\nabla \times H = J_f + \frac{\partial D}{\partial t}$

关键补充与推导
#

  1. 高斯定律与库仑定律关联：$|E| = \frac{1}{4\pi\varepsilon_0} \frac{|q|}{r^2}$，描述电场与电荷的关系。
  2. 磁高斯定律：$\nabla \cdot B = 0$，表明磁场无孤立磁荷，可表示为矢势的旋度$B = \nabla \times A$。
  3. 法拉第定律：揭示变化的磁场产生电场，核心是磁通量变化率与感应电场的关联。
  4. 位移电流的引入：麦克斯韦为解决安培环路定律在非稳恒电流下的矛盾，引入位移电流$J_D = \frac{\partial D}{\partial t}$，保证电荷守恒（$\nabla \cdot J + \frac{\partial \rho}{\partial t} = 0$）。

均匀各向同性介质中的波动方程
#

  • 由麦克斯韦方程组推导得到：$\nabla^2 D - \mu\varepsilon \frac{\partial^2 D}{\partial t^2} = 0$，$\nabla^2 H - \mu\varepsilon \frac{\partial^2 H}{\partial t^2} = 0$。
  • 解的形式：$\psi = A e^{i(\omega t - k \cdot r)}$，其中波数$|k| = \omega\sqrt{\mu\varepsilon}$，相速度$v = \frac{1}{\sqrt{\mu\varepsilon}}$。
  • 自由空间中：$v = c = \frac{1}{\sqrt{\mu_0\varepsilon_0}} = 299792458 , \text{m/s}$。

含磁单极子的麦克斯韦方程组（拓展）
#

无磁单极子 含磁单极子（韦伯规范） 含磁单极子（安培·米规范）
$\nabla \cdot E = \frac{\rho_e}{\varepsilon_0}$ $\nabla \cdot E = \frac{\rho_e}{\varepsilon_0}$ $\nabla \cdot E = \frac{\rho_e}{\varepsilon_0}$
$\nabla \cdot B = 0$ $\nabla \cdot B = \rho_m$ $\nabla \cdot B = \mu_0\rho_m$
$\nabla \times E = -\frac{\partial B}{\partial t}$ $\nabla \times E = -\frac{\partial B}{\partial t} + J_m$ $\nabla \times E = -\frac{\partial B}{\partial t} + \mu_0 J_m$
$\nabla \times B = \mu_0\varepsilon_0 \frac{\partial E}{\partial t} + \mu_0 J_c$ $\nabla \times B = \mu_0\varepsilon_0 \frac{\partial E}{\partial t} + \mu_0 J_c$ $\nabla \times B = \mu_0\varepsilon_0 \frac{\partial E}{\partial t} + \mu_0 J_c$
$F = q_e(E + v \times B)$ $F = q_e(E + v \times B) + \frac{q_m}{\mu_0}(B - v \times \frac{E}{c^2})$ $F = q_e(E + v \times B) + q_m(B - v \times \frac{E}{c^2})$

1.3 电磁场的能量与动量密度
#

能量密度与坡印廷矢量
#

  • 能量密度：$U = \frac{1}{2}(E \cdot D + B \cdot H)$，描述电磁场存储的能量。
  • 坡印廷矢量：$S = E \times H$，表示电磁能的能流密度（单位时间通过单位面积的能量）。
  • 能量守恒关系：$\frac{\partial U}{\partial t} + \nabla \cdot S = -J \cdot E$，无电流时（$J=0$）满足$\frac{\partial U}{\partial t} + \nabla \cdot S = 0$。

色散介质中的有效能量密度
#

  • 电场与电位移矢量的频域关系：$D(x, \omega) = \varepsilon(\omega) E(x, \omega)$。
  • 有效能量密度：$u_{eff} = \frac{1}{2} \text{Re}\left[\frac{d(\omega\varepsilon)}{d\omega}(\omega_0) |E(x,t)|^2 + \frac{d(\omega\mu)}{d\omega}(\omega_0) |H(x,t)|^2\right]$，其中$\frac{d(\omega\varepsilon)}{d\omega}(\omega_0) = \varepsilon(\omega_0) + \omega_0 \frac{d\varepsilon}{d\omega}(\omega_0)$。

电磁场的动量
#

  • 动量密度：$g = \frac{1}{c^2} S = \mu_0\varepsilon_0 E \times H$，体现电磁场的粒子性。

1.4 色散与群速度
#

海森堡不确定性原理的影响
#

  • 不存在严格单色波（$\Delta\omega \neq 0$），实际光波为多个单色波的叠加。

两列平面波的叠加
#

  • 叠加场：$U(z,t) = 2A \cos\left[\frac{1}{2}(\Delta\omega t - \Delta k z)\right] e^{-i(\omega_0 t - k_0 z)}$，形成拍频现象，包络的传播速度为群速度。

多列平面波的叠加与群速度推导
#

  • 叠加场的一般形式：$U(z,t) = \frac{1}{2\pi} \int_{-\infty}^{\infty} A(\omega) e^{-i[\omega t - k(\omega) z]} d\omega$。
  • 泰勒展开：$k(\omega) = k(\omega_0) + \left(\frac{dk}{d\omega}\right){\omega_0} (\omega - \omega_0) + \frac{1}{2!}\left(\frac{d^2k}{d\omega^2}\right){\omega_0} (\omega - \omega_0)^2$。
  • 群速度定义：$v_g = \left(\frac{d\omega}{dk}\right)_{\omega_0}$，为包络的传播速度。

群速度色散（GVD）
#

  • 色散参数：$D = -\frac{2\pi c}{\lambda^2} \frac{d^2k}{d\omega^2}$，描述群速度随频率的变化，导致脉冲展宽。

群速度与能量速度的关系
#

  • 无损耗介质：$v_g = v_e = \frac{c}{n_{eff}}$。
  • 弱损耗介质：$v_g \approx v_e \leq \frac{c}{n_{eff}}$。
  • 强损耗介质：$v_e \leq \frac{c}{n_{eff}}$，群速度无严格上限（不违背相对论，因群速度不传递信息）。

1.5 空间色散与光束展宽
#

海森堡不确定性原理的空间效应
#

  • 不存在严格平面波（$\Delta k \neq 0$），光束的角谱分布导致空间色散。

角谱表示与光束传播
#

  • 角谱函数：$A\left(\frac{\alpha}{\lambda}, \frac{\beta}{\lambda}, 0\right) = \int_{-\infty}^{\infty} U(x,y,0) e^{-j2\pi\left(\frac{\alpha}{\lambda}x + \frac{\beta}{\lambda}y\right)} dxdy$，光束可视为不同方向平面波的叠加。
  • 传播后的角谱：$A\left(\frac{\alpha}{\lambda}, \frac{\beta}{\lambda}, z\right) = A\left(\frac{\alpha}{\lambda}, \frac{\beta}{\lambda}, 0\right) e^{j\frac{2\pi}{\lambda}\sqrt{1-\alpha^2-\beta^2} z}$。

光束展宽的起源
#

  • 动量守恒：$k_z(k_x) = \sqrt{k^2 - k_x^2}$，傍轴近似下$k_z(k_x) = k - \frac{k_x^2}{2k}$。
  • 空间色散：$D = -\frac{\partial^2 k_z}{\partial k_x^2}$，导致光束传播过程中自然展宽。

显微镜分辨率极限（阿贝极限）
#

  • 公式：$d = \frac{\lambda}{2n\sin\theta}$，其中$n$为介质折射率，$\theta$为物镜孔径角。
  • 物理本质：由光的衍射（空间色散）决定，与海森堡不确定性原理一致（$\Delta x \cdot \Delta p \geq \hbar$）。

1.6 费马原理、反射与折射
#

费马原理的两种形式
#

  1. 原始形式（最小时间原理）：光线从A到B的传播路径是耗时最短的路径，$t = \int_{A}^{B} \frac{ds}{v} = \frac{1}{c} \int_{A}^{B} n ds$。
  2. 现代形式（稳定光程原理）：光线的光程取极值（极小、极大或恒定），$\delta S = \delta \int_{A}^{B} n(r) dr = 0$。

折射定律（斯涅尔定律）的推导
#

  • 由最小时间原理：$\frac{dt}{dx} = \frac{\sin\theta_1}{v_1} - \frac{\sin\theta_2}{v_2} = 0$，结合$v = \frac{c}{n}$，得$\frac{\sin\theta_1}{\sin\theta_2} = \frac{n_2}{n_1}$。

相关原理拓展
#

  1. 惠更斯原理：波前上各点为子波源，新波前为子波的包络，可解释衍射、反射和折射现象。
  2. 广义斯涅尔定律：适用于超构表面，考虑相位突变，$n_1\sin\theta_1 - n_2\sin\theta_2 = \frac{dp(n)}{d\omega}$。

1.7 电磁场的偏振
#

平面波的偏振表示
#

  • 平面波电场：$E = A e^{i(\omega t - k \cdot r)}$，满足横波条件$k \cdot E = 0$。
  • 正交分量表示：$E_x = A_x \cos(\omega t + \delta_x)$，$E_y = A_y \cos(\omega t + \delta_y)$，相位差$\delta = \delta_y - \delta_x$。

偏振态分类
#

  1. 线偏振：$\delta = 0$或$\pi$，电场矢量沿固定方向振动，$\frac{E_y}{E_x} = \pm \frac{A_y}{A_x}$。
  2. 圆偏振：$A_x = A_y$，$\delta = \pm \frac{\pi}{2}$：
    • 右旋圆偏振（RHC）：$\delta = -\frac{\pi}{2}$；
    • 左旋圆偏振（LHC）：$\delta = \frac{\pi}{2}$。
  3. 椭圆偏振：$\delta \neq 0, \pm\frac{\pi}{2}, \pi$，电场矢量轨迹为椭圆，满足$\left(\frac{E_x}{A_x}\right)^2 + \left(\frac{E_y}{A_y}\right)^2 - 2\frac{\cos\delta}{A_x A_y} E_x E_y = \sin^2\delta$。

偏振态的数学表征
#

  1. 琼斯矢量：描述完全偏振光，例如：
    • 水平线偏振：$\begin{pmatrix} 1 \ 0 \end{pmatrix}$；
    • 垂直线偏振：$\begin{pmatrix} 0 \ 1 \end{pmatrix}$；
    • 右旋圆偏振：$\frac{1}{\sqrt{2}}\begin{pmatrix} 1 \ -i \end{pmatrix}$。
  2. 琼斯矩阵：描述光学元件对偏振态的作用，例如：
    • 水平偏振片：$\begin{pmatrix} 1 & 0 \ 0 & 0 \end{pmatrix}$；
    • 四分之一波片（快轴垂直）：$e^{i\pi/4}\begin{pmatrix} 1 & 0 \ 0 & i \end{pmatrix}$。
  3. 庞加莱球：几何表征偏振态，球面上任一点对应一种完全偏振光，参数与斯托克斯矢量关联：$S_0 = 1$，$S_1 = \cos2\psi$，$S_2 = \sin2\psi\cos\delta$，$S_3 = \sin2\psi\sin\delta$。

1.8 旋量与潘查拉特纳姆相位
#

旋量的数学描述
#

  • 旋量形式：$s = \sqrt{r} e^{-i\alpha/2} \begin{pmatrix} \cos(\theta/2) e^{-i\phi/2} \ \sin(\theta/2) e^{i\phi/2} \end{pmatrix}$，可理解为“几何的平方根”。
  • 旋转特性：绕z轴旋转$\theta_r$时，旋量变换为$\begin{pmatrix} e^{-i\theta_r/2} & 0 \ 0 & e^{i\theta_r/2} \end{pmatrix} \begin{pmatrix} a \ b \end{pmatrix}$，存在角加倍效应。

潘查拉特纳姆相位（几何相位）
#

  • 定义：偏振光在庞加莱球上沿闭合路径演化时产生的额外相位，源于偏振态的几何演化，与传播路径无关。
  • 应用：超构材料中的自旋选择传输、几何相位导光等。

1.9 平行输运及其应用
#

列维-奇维塔平行输运
#

  • 欧氏平面：平行输运后矢量方向不变。
  • 弯曲表面（如球面）：沿闭合路径平行输运后矢量产生转角，转角满足高斯-博内定理$\varphi = \int_{\Omega} G d\Omega = \frac{S(\Omega)}{R^2}$（$G$为高斯曲率，$S(\Omega)$为区域面积）。

应用场景
#

  1. 傅科摆：地球自转导致摆平面旋转，本质是平行输运在球面（地球表面）的体现，转角$\alpha = 2\pi(1 - \sin\theta)$（$\theta$为纬度）。
  2. 电磁波偏振旋转：包括法拉第旋转（磁场作用下的偏振旋转，$\theta = V d H$，$V$为费尔德常数）和几何相位导致的偏振旋转。

1.10 麦克斯韦方程组的原始形式与拓展
#

原始麦克斯韦方程组（矢量势形式）
#

  • 矢势与场强关系：$E = -\nabla\phi - \frac{\partial A}{\partial t}$，$B = \nabla \times A$。
  • 原始方程组以矢势和标势为核心，后续被亥维赛-赫兹形式（场强形式）简化，但矢势的物理意义不可忽视。

矢势的物理意义
#

  • 阿哈罗诺夫-玻姆效应：即使电场$E=0$、磁场$B=0$，矢势$A$仍能产生可观测的物理效应，证明矢势是真实的物理存在，而非单纯的数学工具。
  • 动量修正：带电粒子的正则动量为$p + qA$，而非机械动量$p$，体现矢势的动量贡献。

电磁学的几何化
#

  • 外尔的规范理论：尝试将电磁学几何化，认为矢量长度的“规范变换”对应电磁相互作用，虽最初遭爱因斯坦反对，但量子力学中得以验证（与相位因子关联）。

1.11 电磁波的波粒二象性
#

核心特性
#

  • 波动性：表现为干涉、衍射、偏振等现象，由麦克斯韦方程组描述。
  • 粒子性：表现为光电效应、康普顿散射等现象，光子能量$E = \hbar\omega$，动量$p = \hbar k$。

量子力学诠释
#

  1. 哥本哈根诠释：测量前粒子处于叠加态（如双缝干涉中光子同时通过两缝），测量导致波函数坍缩，呈现粒子性。
  2. 关键名言与观点：
    • 爱因斯坦：“五十年来的深思熟虑，并未让我更接近‘光量子是什么’的答案。”
    • 玻尔：“深奥的真理的对立面也是深奥的真理。”
    • 海森堡：“我们观察的不是自然本身，而是自然向我们的探寻方法所暴露的一面。”

第2章 光学谐振腔与粒子散射
#

2.1 光学谐振腔简介
#

核心定义与谐振条件
#

  • 光学谐振腔是限制光场在特定空间内振荡、增强光与物质相互作用的装置，核心满足玻尔谐振条件：$\delta \phi = 2m\pi$（$m$为整数），即光在腔内往返一周的相位变化为$2\pi$的整数倍。
  • 几何关联：谐振条件本质是腔内光程需包含整数个波长，即$\beta k(r)dr = 2m\pi$（$\beta$为传播常数，$k(r)$为局域波数）。

关键参数
#

  1. 品质因数（$Q$）
    • 定义：$Q = \omega \times \frac{\text{谐振腔存储的场能量}}{\text{谐振腔耗散的功率}}$，反映腔的储能能力与损耗特性。
    • 能量存储：电场能量$U_{electric} = \frac{1}{8}\varepsilon E_0^2 AL$，磁场能量$U_{magnetic} = U_{electric}$，总储能$U = \frac{1}{4}\varepsilon E_0^2 V$（$V$为腔体积）。
  2. 模式密度
    • 波矢关系：$k_x^2 + k_y^2 + k_z^2 = \left(\frac{\omega n}{c}\right)^2$，谐振波矢满足$k_x = \frac{r\pi}{a}$、$k_y = \frac{s\pi}{b}$、$k_z = \frac{t\pi}{c}$（$r,s,t$为整数，$a,b,c$为腔尺寸）。
    • 模式数与模式密度：
      • 0到$k$的总模式数：$N(k) = \frac{k^3 V}{6\pi^2}$；
      • 频率$v$附近的模式密度：$p(v) = \frac{8\pi v^2 n^3 V}{c^3}$（考虑偏振）；
      • 色散介质中：$p(v) = \frac{3\pi v^2 n^2 V\left(n + v\frac{\partial n}{\partial v}\right)}{c^3}$。

波导与谐振腔的关联
#

  • 平板波导通过两端反射形成谐振腔，模式由波导的横向约束与腔的纵向谐振共同决定，不同模式（TE/TM）的有效折射率随腔参数（如厚度）呈现离散分布。

2.2 法布里-珀罗（Fabry-Perot）谐振腔
#

结构与光程差
#

  • 结构：由两块平行反射镜组成，中间填充折射率为$n$的介质，腔长为$l$。
  • 光程差：$\Delta L = 2l\cos\theta$（$\theta$为入射光与腔轴夹角），相位差$\delta = \frac{4\pi n l \cos\theta}{\lambda}$。

反射与透射特性
#

  1. 斯托克斯关系（互易与时间反转对称性）：$t_{12}t_{21} + r_{12}r_{21} = 1$，$t_{12}r_{21} + r_{12}t_{12} = 0$（$t$为透射系数，$r$为反射系数）。
  2. 反射率与透射率：
    • 反射率：$\frac{I_r}{I_i} = \frac{4R\sin^2(\delta/2)}{(1-R)^2 + 4R\sin^2(\delta/2)}$；
    • 透射率：$\frac{I_t}{I_i} = \frac{(1-R)^2}{(1-R)^2 + 4R\sin^2(\delta/2)}$（$R = r^2$为反射镜反射率）。
  3. 谐振特性：
    • 谐振条件：$\delta = 2m\pi$，对应透射率最大（$I_t/I_i = 1$），谐振频率$v_m = m\frac{c}{2nl\cos\theta}$；
    • 自由光谱范围：$\Delta v = v_{m+1} - v_m = \frac{c}{2nl\cos\theta}$，色散介质中$\Delta v = \frac{c}{2n_g l\cos\theta}$（$n_g$为群折射率）。

腔内场与群延迟
#

  • 腔内平均光强：$I_{ave} = \frac{1+R}{1-R}I_i$，高反射率（如$R=0.99$）时腔内光强显著增强（$I_{ave}=199I_i$）。
  • 增强群延迟：谐振时群延迟$\tau_g = \frac{1+R}{1-R}\tau_0$（$\tau_0$为光在腔内的单程传播时间），源于光在反射镜间的多次反射叠加。

应用示例
#

  • 光谱分析：利用高分辨率的透射峰分辨精细光谱；
  • 激光谐振腔：作为激光器的核心部件，实现光的振荡与放大；
  • 引力波探测：LIGO装置利用法布里-珀罗腔的高灵敏度测量时空形变。

2.3 谐振腔的损耗与线宽
#

损耗机制
#

  1. 非理想反射：反射镜反射率$R<1$导致透射损耗；
  2. 吸收与散射：腔内介质对光的吸收、散射损耗；
  3. 衍射损耗：光场超出反射镜范围导致的辐射损耗。

模式寿命与线宽
#

  1. 模式寿命（$\tau_c$）：描述腔内能量衰减的时间，$ \frac{d\mathscr{E}}{dt} = -\frac{\mathscr{E}}{\tau_c}$，$\mathscr{E}(t) = \mathscr{E}(0)e^{-t/\tau_c}$。
    • 法布里-珀罗腔寿命：$\tau_c = \frac{n}{c\left[\alpha - \frac{1}{l}\ln\sqrt{R_1R_2}\right]}$（$\alpha$为介质吸收系数，$R_1,R_2$为两镜反射率）。
  2. 品质因数与线宽关联：$Q = \omega_0\tau_c$，谐振线宽$\Delta v = \frac{1}{2\pi\tau_c} = \frac{v_0}{Q}$，损耗越大，$Q$越小，线宽越宽。

相干完美吸收（CPA）
#

  • 定义：当满足特定相位条件时，入射光被完全吸收（$T=0$），源于入射光与散射光的相消干涉。
  • 条件：$R_{resonance} = \frac{(\sqrt{R_1} - A)^2}{(1 - \sqrt{R_1}A)^2}$（$A=e^{-\alpha L}$为介质透过率），当$R_1=A$时实现完全吸收。

2.4 多腔谐振腔与光子晶体
#

多层膜与增透膜
#

  • 结构：由多层不同折射率的介质组成，本质是多个子腔的叠加。
  • 增透膜条件：$n_2 = \sqrt{n_1n_3}$且$\phi = \pi/2, 3\pi/2,…$（$\phi = 2\pi n_2 d/\lambda$为光在膜内的相位变化），此时反射率$r=0$，实现完美透射。

布拉格反射镜
#

  • 结构：由折射率交替变化的多层介质组成，每层厚度为$\lambda/4$。
  • 反射特性：利用各层反射光的相长干涉，实现特定波长的高反射率，反射率随层数增加而提高。

光子晶体
#

  • 定义：折射率在空间周期性变化的结构，形成光子带隙，禁止特定频率的光传播。
  • 应用：光子晶体光纤、光子晶体激光器、慢光器件等，利用带隙特性实现光的强约束与调控。

安德森局域化
#

  • 无序介质中的光散射导致光被局域在有限空间内，形成随机激光，本质是无序诱导的“随机谐振腔”。

2.5 粒子散射基础
#

散射的基本概念
#

  • 过程：入射光与粒子相互作用后，能量被重新分布（散射+吸收），总消光=散射+吸收（$W_{ext}=W_{sca}+W_{abs}$）。
  • 边界条件：粒子表面电场、磁场的切向分量连续，即$\left(E_2 - E_1\right) \times \hat{n}=0$，$\left(H_2 - H_1\right) \times \hat{n}=0$（$\hat{n}$为表面法向量）。

关键物理量
#

  1. 截面参数：

    • 散射截面$C_{sca} = \frac{W_{sca}}{I_i}$，吸收截面$C_{abs} = \frac{W_{abs}}{I_i}$，消光截面$C_{ext} = C_{sca}+C_{abs}$；
    • 微分散射截面$\frac{dC_{sca}}{d\Omega} = \frac{|f(\theta,\phi)|^2}{k^2}$（$f$为散射振幅，$\Omega$为立体角）；
    • 效率因子：$Q_{sca}=\frac{C_{sca}}{A}$、$Q_{abs}=\frac{C_{abs}}{A}$、$Q_{ext}=\frac{C_{ext}}{A}$（$A$为粒子几何截面）。

  2. 光学定理：$C_{ext} = \frac{4\pi}{k}\text{Re}[f(0)]$，消光截面仅由前向散射振幅的实部决定，揭示了前向散射与总消光的关联。

粒子与平板的散射类比
#

  • 粒子散射可类比为“三维平板”的散射，入射光在粒子表面反射、折射后形成散射光，内部存在折射光场（局域场）。

2.6 米氏（Mie）理论：球形粒子散射
#

理论基础
#

  • 适用范围：任意尺寸的球形粒子（小到瑞利散射极限，大到几何光学极限），需求解球坐标系下的麦克斯韦方程组。
  • 场展开：
    • 入射场、散射场、粒子内场均展开为球谐函数与球汉克尔函数的叠加；
    • 球坐标系下波动方程：$\nabla^2\psi + k^2\psi=0$，解为$\psi(r,\theta,\phi)=R(r)\Theta(\theta)\Phi(\phi)$（$R$为径向函数，$\Theta$为角向函数，$\Phi$为方位角函数）。

关键函数与系数
#

  1. 角向函数：关联勒让德函数$P_n^m(\cos\theta)$，描述场的角分布；
  2. 径向函数：球贝塞尔函数$j_n(\rho)$、球诺依曼函数$y_n(\rho)$、球汉克尔函数$h_n^{(1)}(\rho)=j_n(\rho)+iy_n(\rho)$（$\rho=kr$）；
  3. 散射系数（$a_n,b_n$）：由边界条件推导，决定各阶多极子（偶极子、四极子等）的散射贡献。

散射特性
#

  1. 多极子散射：
    • 小粒子（$kr\ll1$）：以电偶极子、磁偶极子散射为主，过渡到瑞利散射；
    • 大粒子（$kr\gg1$）：高阶多极子贡献显著，散射图案呈现多瓣结构，趋近于几何光学散射。
  2. 瑞利散射（米氏理论的特例）：
    • 条件：$kr\ll1$，$a_1\approx-\frac{2x^3}{3}\frac{m^2-1}{m^2+2}$（$x=kr$，$m=n_1/n_0$为粒子相对折射率）；
    • 散射光强：$I_s = \frac{8\pi^4 N a^6}{\lambda^4 r^2}\left|\frac{m^2-1}{m^2+2}\right|^2(1+\cos^2\theta)I_i$，与$\lambda^4$成反比，解释天空呈蓝色、日出日落呈红色的现象。

金属粒子散射
#

  • 表面等离激元共振：金属粒子中自由电子集体振荡，导致散射截面在特定波长处出现尖锐峰值；
  • 应用：表面增强拉曼散射（SERS）、等离子体光热治疗（如癌症治疗中的纳米壳层）。

2.7 金属粒子谐振的几何诠释
#

玻尔条件的延伸
#

  • 金属粒子的谐振满足$\beta k(r)dr = 2m\pi$，其中$k_{spp} = \frac{2\pi}{\lambda}\sqrt{\frac{\varepsilon_m\varepsilon_d}{\varepsilon_m+\varepsilon_d}}$（$\varepsilon_m$为金属介电常数，$\varepsilon_d$为周围介质介电常数）为表面等离激元波数。
  • 模式分类：偶极子（$m=1$）、四极子（$m=2$）、六极子（$m=3$）等，高阶模式对应更高频率。

谐振特性的几何影响
#

  1. 尺寸依赖：粒子尺寸减小，谐振频率蓝移；尺寸增大，谐振频率红移；
  2. 模式品质因数：高阶模式的$Q$值更高，损耗更小；
  3. 色散特性：
    • 正常色散（$d\omega/dk>0$）：对应常规散射；
    • 平带（$d\omega/dk=0$）：模式重叠，增强局域场；
    • 反常色散（$d\omega/dk<0$）：出现反向模式，导致异常散射。

2.8 松弛全反射（Relaxed TIR）
#

原理与条件
#

  • 传统全反射：$n_1>n_2$，光被约束在介质1中，存在倏逝波；
  • 松弛全反射：通过各向异性介质或亚波长结构，放松$n_1>n_2$的条件，仍可实现光的强约束（$|E_z|\gg1$）。
  • 约束条件：$\frac{x_1^2}{\varepsilon_1} + \frac{x_2^2}{\varepsilon_2} = k_0^2$，利用介电常数的各向异性实现亚衍射极限的光约束。

应用
#

  • 透明亚衍射光学器件：无需金属即可实现纳米尺度的光 confinement，应用于高分辨率成像、集成光学器件。
